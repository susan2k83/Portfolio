About Me

I am a Front End Web Developer living in North San Diego county but my native ties are to the ever bright skies and warm sands of Palm Springs. I enjoy the creativity that front end web development offers... the digital representation of the imagination.

A graduate from a full-stack Web Development program, I am currently working towards an internship full or part time in front-end web development. Experienced in HTML5, CSS3, Javascript, JQuery, Bootstrap, WordPress, PHP for Wordpress. Additionally, knowledgeable in elementary education, the sciences and healthcare fields

Applications Built

BandWagon

Using SoundCloud and Wkikpedia BandWagon is an app that allows users to select a band or artist and view related facts and listen to music tracks.

TED2

TED2 is a simplified navigational tool used to access and search the TED database.

WanderSpark

WanderSpark is digital travel planner designed to assist a user in selecting and organizing points of interest at their destination of choice.

Simple Statistics Helper

This one page application features a quick tool to sort a list of numbers in ascending order and return the mean mode and median.

Modal Nodes Fan Page

The Modal Nodes Fan Page began as a small class project featuring one page of html and a bit of css. It has since been rebuilt using Bootstrap 4 and highlights the fictional cantina band from Star Wars, "Figrin D'an and The Modal Nodes"

Birthday Finder

Birthday Finder is a simple app that allows a user to input a bithdate, which then returns the day of the week that corresponds to that date.

Architectural Research Consulting

Architectural Research Consulting is a science-based architectural firm specializing in clean room and laboratory planning. This is website is hosted by WIX and utilizes a simple template that was extensively customized to suit the clients needs.

Life In Pictures

Life In Pictures is a portfolio project for a client looking to have a quick way to access, display and share photos. This is an evolving project using Materialize CSS and whose format may be subject to change as more photos are added.
